# Roadmap — 版本规划

## v0.1.0 — 项目脚手架与基础框架

**目标**：搭建完整的开发环境，跑通前后端通信。

- [x] Tauri 2 项目初始化（`npm create tauri-app`）
- [x] React + TypeScript + Vite 前端脚手架
- [x] Tailwind CSS + shadcn/ui 配置
- [x] SQLite 数据库初始化与基础 CRUD
- [x] Tauri IPC 命令基础封装
- [x] 基础窗口配置（尺寸、位置、无边框）

## v0.2.0 — 剪贴板核心功能

**目标**：实现剪贴板监听、记录、展示的核心链路。

- [x] Rust 剪贴板监听器（文本类型）
- [x] 内容去重（hash 对比）
- [x] 数据库写入与读取
- [x] 前端历史记录列表展示
- [x] 点击记录复制回剪贴板
- [x] 前后端事件通信（新记录通知）

## v0.3.0 — 图片支持

**目标**：完整支持图片类型的剪贴板记录。

- [x] 图片剪贴板内容读取
- [x] 图片压缩与缩略图生成
- [x] 图片 Blob 存储
- [x] 前端图片缩略图展示
- [x] 图片预览弹窗（查看原图）

## v0.4.0 — 搜索与过滤

**目标**：实现快速查找历史内容的能力。

- [x] 搜索栏 UI
- [x] 文本内容模糊搜索
- [x] 按类型过滤（文本/图片）
- [x] 搜索防抖优化
- [x] 高亮搜索关键词

## v0.5.0 — 系统集成

**目标**：完成桌面应用的核心系统级体验。

- [x] 系统托盘图标与菜单
- [x] 全局快捷键唤起窗口
- [x] 窗口失焦自动隐藏
- [x] 开机自启动
- [x] macOS 菜单栏集成

## v0.6.0 — 设置与个性化

**目标**：让用户可以自定义应用行为。

- [x] 设置面板 UI
- [x] 历史记录数量限制配置
- [x] 快捷键自定义
- [x] 主题切换（亮色/暗色/跟随系统）
- [x] 数据清理（手动/自动）

## v0.7.0 — 体验打磨

**目标**：优化细节，提升整体体验。

- [x] 列表虚拟滚动（大数据量性能）
- [x] 记录置顶/收藏功能
- [x] 批量删除
- [x] 键盘导航（上下选择、回车确认）
- [x] 操作反馈动画
- [x] 应用图标设计

## v0.8.0 — 高级功能

**目标**：增加差异化功能，提升工具价值。

- [ ] 智能分类（链接、代码、邮件等自动识别）
- [ ] 历史记录分组（按日期）
- [ ] 导出/导入历史数据
- [ ] 跨设备同步（可选，探索阶段）

## v1.0.0 — 正式发布

**目标**：稳定版发布，完整文档，支持双平台分发。

- [ ] 全面测试（单元测试 + 集成测试）
- [ ] 性能优化与内存泄漏排查
- [ ] macOS 公证（Notarization）
- [ ] Windows 代码签名
- [ ] 自动更新机制
- [ ] GitHub Actions CI/CD
- [ ] 官方网站 / 落地页
- [ ] 用户文档
