# Roadmap — 版本规划

> 当前版本：**v0.1.4**（2026-05-17）

## v0.1.0 — 首个正式版本（已发布 2026-05-15）

**目标**：完成剪贴板管理工具的核心功能，支持双平台分发。

- [x] Tauri 2 + React 18 + TypeScript + Vite 项目脚手架
- [x] Tailwind CSS + shadcn/ui 配置
- [x] SQLite 数据库层（rusqlite + r2d2 连接池）
- [x] Tauri IPC 命令封装
- [x] Rust 剪贴板监听器（文本 + 图片）
- [x] 内容去重（SHA256 hash）
- [x] 前端历史记录列表展示
- [x] 图片压缩缩略图与预览
- [x] 搜索栏、模糊搜索、类型过滤、关键词高亮
- [x] 系统托盘、全局快捷键、窗口自动隐藏
- [x] 开机自启动、macOS 菜单栏
- [x] 托盘历史菜单（最近 10 条，点击复制）
- [x] 设置面板（主题、历史数量、数据清理）
- [x] 中英文切换
- [x] 虚拟滚动、置顶收藏、批量删除
- [x] 键盘导航、操作反馈动画
- [x] 官方网站（Next.js）
- [x] GitHub Actions CI/CD（macOS ARM/Intel + Windows）

- [x] 启用自动更新功能（签名验证）

## v0.2.0 — 体验优化

**目标**：优化现有功能体验，修复已知问题。

- [ ] 快捷键自定义
- [ ] 历史记录分组（按日期）
- [ ] 导出/导入历史数据
- [ ] 更多内容类型支持（链接、代码片段自动识别）

## v0.3.0 — 高级功能

**目标**：增加差异化功能，提升工具价值。

- [ ] 智能分类（链接、代码、邮件等自动识别）
- [ ] 跨设备同步（可选，探索阶段）

## v1.0.0 — 正式稳定版

**目标**：稳定版发布，完整文档，企业级质量。

- [ ] 全面测试（单元测试 + 集成测试）
- [ ] 性能优化与内存泄漏排查
- [ ] macOS 公证（Notarization）
- [ ] Windows 代码签名
- [ ] 用户文档
