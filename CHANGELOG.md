# Changelog

本项目遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 格式，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [0.1.4] - 2026-05-17

### 修复
- 禁用 WebView 默认右键菜单（打印、刷新、Reload 等）
- 托盘菜单历史记录文本截断长度限制，避免菜单过宽

## [0.1.3] - 2026-05-16

### 新增
- 启用自动更新功能（签名验证）

## [0.1.2] - 2026-05-16

### 修复
- 修复 Windows 顶部显示英文应用菜单（About、Hide、Edit）
- NSIS 安装界面添加中文支持

## [0.1.1] - 2026-05-15

### 修复
- 修复 Windows 运行时弹出终端窗口（设置 windows subsystem）
- 修复虚拟滚动列表项重叠（按内容类型动态估算高度）
- 更新 macOS 安装说明，补充"已损坏"提示解决方案
- CI release 产物精简：去掉 MSI，上传通用文件名
- 官方网站下载页自动检测 macOS 架构（ARM/Intel）

## [0.1.0] - 2026-05-15

首个正式版本，包含完整的剪贴板管理核心功能。

### 新增
- **剪贴板监听**：Rust 后台线程 200ms 轮询，实时监听文本和图片剪贴板变化
- **内容去重**：SHA256 hash 对比，避免重复记录
- **历史记录管理**：搜索、过滤、置顶收藏、批量删除
- **图片支持**：图片剪贴板读取、压缩缩略图、前端展示与预览
- **搜索与过滤**：搜索栏 UI、模糊搜索（200ms 防抖）、类型过滤、关键词高亮
- **系统集成**：系统托盘图标与菜单、全局快捷键（Alt+Shift+V）、窗口失焦自动隐藏、开机自启动
- **macOS 菜单栏**：原生应用菜单 + 编辑菜单
- **托盘历史菜单**：右键托盘展示最近 10 条历史，点击即复制，置顶项优先
- **设置面板**：主题切换（浅色/深色/跟随系统）、历史数量配置、数据清理
- **中英文切换**：React Context + Hook 轻量 i18n，默认中文，支持跟随系统
- **体验优化**：虚拟滚动（@tanstack/react-virtual）、键盘导航、操作反馈动画
- **UI 优化**：无标题栏设计，设置按钮移至搜索栏右侧

### 技术栈
- Tauri 2 + React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- SQLite (rusqlite) + Zustand
- 支持 macOS (ARM/Intel) + Windows
